Програма

Become a successful data analyst

Студентите ќе научат:

  • 1. What is data analytics?
  • 2. How can data analytics help me?
  • 3. What are the requirements to be successful data analyst?
  • 4. Which tools I need for data analysis?
  • 5. What can I do in Excel?
  • 6. What can I do with SQL?

Overview of data analysis, requirements, tools, approach to a business stakeholder

Start small, do better in Excel

Студентите ќе научат:

  • 1. Learn the Excel's smart functions.
  • 2. Extend the insights with in-built tools.

Excel's basic and advanced functions / tools, connections with files, tables, charts, pivots, power-pivots, data models

Extend the possibilities with SQL

Студентите ќе научат:

  • 1. Why SQL?
  • 2. How to find the required data?
  • 3. Can I use it in various development environments?
  • 4. What about optimizations, aggregations, smart calculations?
  • 5. What about the code backup and versioning?

Syntax, possibilites, joins, groups, aggregations, functions, procedures, data mining, data cleaning, data quality, git

Work with data, be efficient (Data Lake, ETL, Data Marts / Data Warehouse)

Студентите ќе научат:

  • 1. Get the data from different database solutions, but also any kind of file format.
  • 2. Ingest the data in Data Lake.
  • 3. Work with the data and transform it for your needs.
  • 4. Put the unstructured into structured Data Marts / Warehouse.
  • 5. How to mix offline and online data storage?

On-prem and cloud OLTP-OLAP RDS / DW solutions, star / snowflake / galaxy schemas, normalized vs denormalized tables, working with DBs / files (CSV, JSON, XML, flat), ETL / ELT

Get the insights and deliver data-driven decisions

Студентите ќе научат:

  • 1. I have the data, how to make a decision?
  • 2. Can I be alarmed before-hand?
  • 3. What about predictions, trends and outliers?

Work with UseCases, design the dataflow, BI, notifications, alarms, KPIs, ratios, ... (all from the point of final data output, not Power BI or other reporting solution)

  • Cardinality / Relationships
