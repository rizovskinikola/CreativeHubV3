Академија за Software Testing

Почеток на програмата: Февруари 2024

Пријави се по Super Early Bird до 30.11

5 месеци до јуниор тестер.

Потребно предзнаење: Почетник
(Препорачуваме најпрво да го погледнеш бесплатниот курс)

Структура на програмата:
5 месеци програма + Програма за вработување

Формула за Успех

80 часа работа со инструктор

Академијата започнува со учење од искуственото знаење од нашите предавачи. Не е потребно предзнаење бидејќи предавачите ќе започнат од основите. Поставувај прашања, учествувај во вежби и добиј индивидуален фидбек за се она што ќе сработиш.

Програма за вработување

Аплицирај за првата работна понуда во 1-3 месеци по завршувањето на Академијата.

3 практични проекти

Modules

Introduction to Scrum and Agile ways of work
Introduction Week
  • Learn the concept of Scrum
  • Learn the concept of Agile
Week 1: Introduction to Software Testing
  • Overview of software testing
  • Importance of software testing
  • Software development life cycle ( SDLC )
  • Different types of testing ( functional, non-functional, etc. )
  • Introduction to manual testing
Week 2: Manual Testing Fundamentals
  • Acceptance criteria
  • Test case design techniques
  • Test planning and documentation
  • Test execution and defect/bug reporting
  • Test management tools

Hands-on practice: Manual test case creation and execution

Week 3-6: Introduction to OOP ( Java )

Key concepts: classes, objects, inheritance, polymorphism, encapsulation

  • Introduction to the Java programming language (or any other OOP language of choice)
  • Defining classes and objects
  • Class properties (fields) and methods
  • Constructors and object initialization
  • Access modifiers (public, private, protected)

Hands-on practice: Creating and working with classes and objects

  • Inheritance hierarchy and relationships
  • Extending classes and creating subclasses
  • Method overriding and polymorphic behavior
  • Abstract classes and interfaces

Hands-on practice: Implementing inheritance and polymorphism

  • Encapsulation principles
  • Getters and setters for data access
  • Accessor and mutator methods
  • Abstraction and interface design

Hands-on practice: Applying encapsulation and abstraction concepts

Week 7: Introduction to Automation Testing
  • Introduction to automation testing
  • Benefits and challenges of automation testing
  • Test automation frameworks (structure)
  • Popular automation tools (e.g., Selenium, Appium)
  • Back end (API) testing vs Front end testing
  • Unit/testing
Week 8-10: Back-end Test Automation using rest assured ( Java ) - test classes, models, client etc.
  • What is rest API?
  • What is rest assured?
  • How to connect and test a service (API)?
  • What is API?
  • Test classes
  • Models
  • Structure of a framework
  • Creating an automated test

Hands-on practice: Create framework with automated tests (back end)

Week 11-13: Front-end Test Automation using Selenium ( Java ) - GUI
  • What is GUI (graphical user interface)?
  • What is selenium?
  • Selectors
  • POM (page object model) structure

Hands-on practice: Create framework with automated tests (front end)

Week 14: Database connection - Postgre
  • Types of database
  • How to connect to a database?
  • How to view database?
  • How to connect a database with a automation testing framework?
  • Introduction in Postgre database
Week 15: End 2 End Testing
  • Testing a system as a whole ( back-end + front-end + database )
Week 16: Testing Best Practices
  • Code refactoring
  • Flaky tests
  • Testing reports
  • Other practices
Week 17: Prepare for Job Interview
  • Interview questions for QA Tester
  • Project ideas
  • How to land your first job

ЕУ Диплома со ЕКТС кредити
Се издава на студенти од ЕУ Сертифицирана програма и студенти на Забрзаната програма

  • ЕУ додаток на диплома со ЕКТС кредити го следи моделот на Европска Комисија за неформално образование.
