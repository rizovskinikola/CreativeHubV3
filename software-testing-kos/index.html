  • Acceptance criteria
  • Test case design techniques

  • Test execution and defect/bug reporting

Key concepts: classes, objects, inheritance, polymorphism, encapsulation

  • Introduction to the Java programming language (or any other OOP language of choice)
  • Defining classes and objects

  • Getters and setters for data access
  • Accessor and mutator methods

Week 7: Introduction to Automation Testing
  • Introduction to automation testing
  • Benefits and challenges of automation testing
  • Test automation frameworks (structure)

  • Unit/testing
Week 8-10: Back-end Test Automation using rest assured ( Java ) - test classes, models, client etc.

  • How to connect and test a service (API)?

  • Test classes
  • Models
  • Structure of a framework
  • Creating an automated test

Hands-on practice: Create framework with automated tests (back end)

Week 11-13: Front-end Test Automation using Selenium ( Java ) - GUI
  • What is GUI (graphical user interface)?
  • What is selenium?
  • Selectors
  • POM (page object model) structure

Hands-on practice: Create framework with automated tests (front end)

  • Types of database

  • Code refactoring

for the students of the EU Certified Program

  • The EU Diploma supplement is following the model of the European Commission for Non-Formal Education.

  • Students who are planning to continue their career in the EU are advised to take this certificate.
  • This EU Diploma is accepted by Universities in Europe as additional credits to the application of the student.
