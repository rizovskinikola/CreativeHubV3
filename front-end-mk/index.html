Напредок во кариерата

Имаш некои основни познавања од програмирање, но сакаш да стекнеш детални познавања од искусни предавачи?

Имаш познавања од сродна професија ( UX UI, data science ) и сметаш дека фронт-енд програмирање ќе ти биде основа за поголема плата и поголема заработка како фриленсер?

Промена на кариера

Не си среќен во кариерата што ја имаш и сакаш да се преквалификуваш во кариера што ќе ти даде финансиска слобода и стабилност?

Познаваш луѓе што работат како програмери и имаш желба и ти да го работиш тоа?

Што после средно образование?

Си завршил/a средно и не знаеш што понатаму?

Сакаш кариера што ќе ти даде слобода да работиш од било каде, ќе биде добро платена и во која ќе можеш континуирано да растеш?

Заглави на Факултет?

Не се откажувај од програмирање само затоашто имаш проблем да положиш некој предмет!

Програмата на Академијата е во времетраење од 12 месеци

1 Обука за градење личен бренд Студентите ќе научат:
  • Како да изградиш профил кој ќе биде лесно достапен за регрутери
  • Оптимизирање на твојот Линкедин профил
  • Тактики за градење на професионален личен бренд
2 HTML Студентите ќе научат:
  • Како работи интернетот (комуникација помеѓу веб страните и серверите)
  • Запознавање со код едитор
  • Структура на HTML елементи
  • Објаснување на атрибути и тагови и нивна правилна употреба
  • Мини веб страна само со HTML
3 CSS Студентите ќе научат:
  • Основи на синтакса
  • CSS селектори
  • CSS бои, позадински бои, фонтови, слики
  • Позиционирање на елементи
  • Pseudo класи и pseudo елементи
  • Flexbox
  • Трансформации и анимации
  • Респонзивен дизајн - media queries
  • Користење на надворешни библиотеки и фонтови (CDN)
  • Примери од пракса
  • Респонзивни веб страни со HTML & CSS
4 Реален проект 1 Студентите ќе изработат:
  • Статичка веб страна
5 Bootstrap Студентите ќе научат:
  • Основи на синтакса
  • Grid системи и grid класи
  • Типографија, слики, бои и позадински бои
  • Класи на елементи и компоненти
  • Компоненти и нивна модификација
  • Комплетни респонзивни веб страни со Bootstrap 4
6 Git Студентите ќе научат:
  • Научи како да го поставиш твоето портфолио на Git
7 JavaScript Студентите ќе научат:
  • JavaScript основи (историја, конзола, comments, linking files defer vs. async)
  • Data Types (основни data types, constants, arithmetic operators)
  • Objects и arrays
  • Properties, methods
  • Loops
  • Functions (scope, arrow functions)
  • DOM
  • Events & Async/Await
  • AJAX
  • Classes and object-oriented programming
  • Closures & Recursions
  • Modern JavaScript
8 TypeScript Студентите ќе изработат:
  • Вовед во дополнителната синтакса која се заснова на JavaScript се со цел да се олесни интеграцијата со едиторите што ќе ни помогне за рано детектирање на грешките
9 Реален проект 2 Студентите ќе изработат:
  • Статичка веб страна
10 React js Студентите ќе научат:
  • Introduction to React and Custom Components
  • JSX
  • Handling events and working with state and use state hook
  • Introduction to forms in React
  • Styling React Components
  • Component children, Conditional rendering and Debugging
  • useEffect and useReducer hooks
  • Introduction and working with Material UI
  • Styling Material UI components
  • Working with React Context and the useContext hook
  • Optimisation of React Applications
  • Client - Server communication and working with APIs
  • Creating Custom Hooks in React
  • Introduction and building multi-page applications with React Router
  • Testing React Applications
  • Securing React applications, working Authentication
  • Building and Deploying React Applications
11 Реален проект 3 Студентите ќе изработат:
  • E-commerce Web Application ( in React )
12 Обука за вработување + Дипломирање Студентите ќе научат:
  • Ќе изработиш CV и ќе го финализираш твоето Git портфолио
  • Ќе научиш како да се претставуваш себеси (Elevator Pitch)
  • Ќе научиш како да користиш Requirements-Skills-Value Framework за да ги истакнеш твоите силни страни
  • Ќе учествуваш на тест интервјуа
  • Ќе почнеш да аплицираш!
  • Техничко интервју (симулација)

Што по завршувањето на Академијата?

Во зависност од програмата која си ја одбрал по завршувањето на финалните проекти ќе продолжиш со Обука за вработување каде ќе добиеш јасни насоки како да апплицираш сам.

Доколку си студент на програмата “Академија + Работа” ќе продолжиш со 3 месечно стражирање и потоа нашиот регрутер ќе те препорача на компании со кои соработуваме.

Веднаш по завршување на предавањата

Вклучи се во 8 часовна обука со инструктор каде:

Ќе изработиш CV и Git портфолио.

Ќе го оптимизираш твојот LinkedIn профил.

Ќе научиш како да се претставиш себеси на интервју за работа.

Ќе научиш како да комуницираш со регрутери.

Ќе учествуваш на тест интервјуа

Ќе почнеш да аплицираш!

Стажирање + Гарантирана Работа

Стекни искуство преку работа како дел од фронт-енд тим.

40 часа месечно стражирање како дел од фронт-енд тим.

Секој ден брифинг и фидбек сесија.

По завршување на стажирањето, работи со нашиот регрутер за да те препорача на компании со кои соработуваме.

Ако нашиот регрутер не ти најде работа во период од 1-3 месеци ќе ти ги рефундираме средствата.
