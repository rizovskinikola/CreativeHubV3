Програма

12 месеци работа, 2 пати неделно предавања, секоја сабота вежби со асистент, 24/7 Slack Community.

HTML - 1 недела
Студентите ќе научат
  • Како работи интернетот ( комуникација помеѓу веб страните и серверите)
  • Запознавање со код едитор
  • Структура на HTML елементи
  • Објаснување на атрибути и тагови и нивна правилна употреба
CSS - 3 недели
Студентите ќе научат
  • Основи на синтакса
  • CSS селектори
  • CSS бои, позадински бои, фонтови, слики
  • Позиционирање на елементи
  • Pseudo класи и pseudo елементи
  • Користење на надворешни библиотеки и фонтови (CDN)
  • Примери од пракса
Bootstrap - 4 недели
Студентите ќе научат
  • Основи на синтакса
  • Grid системи и grid класи
  • Типографија, слики, бои и позадински бои
  • Working with Forms and Inputs
  • Класи на елементите и компоненти
  • Комплетнa респонзивнa веб странa со Bootstrap 4
Git - 2 недели
Студентите ќе научат
  • Креирање сметка
  • Инсталирање GitHub
  • Креирање, преклопување и ажурирање гранки
  • Креирање нов GitHub репозитори
  • Додавање датотеки во Git
  • Проверување разлики
Ajax/jQuery - 2 недели
Студентите ќе научат
  • Употребување селектори и филтери за извлекување информации
  • Креирање, инсертирање и манипулирање со соджина на веб страна
  • Пристапување кон екстерни податоци со AJAX
  • Градење справувач со настани
Датабази -4 недели
Студентите ќе научат
  • Основни концепти на бази на податоци. ER дијаграми
  • Системи за управување со бази на податоци (DBMS)
  • Индексирање, надворешни клучеви, податочни типови
  • Спојување на повеќе табели (SQL Joins)
  • Користење на MySQL
  • Нормализација на табели во бази на податоци
  • CRUD операции со користење на SQL
  • Релации во датабазата
  • Концепти на one-to-one, one-to-many, many-to-many релации
PHP - 16 недели
Студентите ќе научат
  • Што е PHP?
  • Градење динамички веб сајтови
  • Структурно прогамирање во PHP
  • Работење со бази на податоци преку PHP
  • Инсталирање и гонфигурирање на PHP и MySQL
  • Разгледување на типови на податоци
  • Разбирање на наследството на класи и модификаторите за пристап
  • Работа со датотеки и директориуми
  • Пишување прилагодени функции
  • Објектно-ориентирано програмирање во PHP и напредни концепти
  • Design Patterns
  • Дефинирање и употреба на класи, методи и атрибути
  • Закачување на датотеките на сервер преку форми
  • Контролирање на кодот со логички изрази и циклуси
  • Употреба на вградените функции на PHP
  • Имплементирање дизајн модели, factory и strategy
Laravel - 12 недели
Студентите ќе научат
  • Користење на Composer package manager
  • Основни на Laravel
  • Валидирање на форми
  • Имплементација на Blade и Eloquent
  • MVC модел - развој на скалабилно софтверско решение
  • REST API и комуникација со сервер
  • Безбедност на пишаниот код
  • Дебагирање
  • Концепти на архитектура
